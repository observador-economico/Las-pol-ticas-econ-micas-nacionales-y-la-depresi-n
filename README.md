# Las-políticas-económicas-nacionales-y-la-depresión
Una vez analizadas las causas de la crisis y de la depresión expondremos, para terminar, las características esenciales de las políticas nacionales seguidas en algunos países entre 1930 y 1938. 
Once the causes of the crisis and the depression have been analyzed, the essential features of the national policies pursued in several countries between 1930 and 1938 can be set out.
